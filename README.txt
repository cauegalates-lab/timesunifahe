IMPORTANTE — APP SCRIPT:
1. Abra o arquivo app-script-completo.gs deste ZIP.
2. Copie todo o conteúdo.
3. Substitua o script atual no Google Apps Script.
4. Salve.
5. Vá em Implantar > Gerenciar implantações > Editar.
6. Selecione Nova versão e clique em Implantar.

A URL não muda quando a implantação existente é atualizada.
